BeHuman Widget Example

This page demonstrates how to use the BeHuman verification widget.

One-Step Installation

Just add this single script tag to your HTML:

<script src="https://markjcsimmons.github.io/behuman/widget.js"></script>

That's it! A "Verify You Are Human" button will automatically appear in the bottom-right corner of your page.

Try It Now

Look at the bottom-right corner of this page - you should see a blue "Verify You Are Human" button that was automatically created!

Or click this button to test:

Optional: Custom Trigger

If you want to trigger it manually instead of using the auto-created button:

<button onclick="BeHuman.show()">Verify You Are Human</button>

Full Example

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    <h1>Welcome to My Site</h1>
    <button onclick="BeHuman.show()">Verify You Are Human</button>
    
    <script src="https://markjcsimmons.github.io/behuman/widget.js"></script>
</body>
</html>
