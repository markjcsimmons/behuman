BeHuman Widget Example

This page demonstrates how to use the BeHuman verification widget.

How to Add to Your Website

Add this script tag to your HTML:

<script src="https://markjcsimmons.github.io/behuman/widget.js"></script>

How to Trigger the Verification

Call BeHuman.show() to display the verification modal:

<button onclick="BeHuman.show()">Verify You Are Human</button>
